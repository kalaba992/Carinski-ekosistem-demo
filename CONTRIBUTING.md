# Contributing

Thank you for reviewing the Carinski Ekosistem demo. We welcome feedback in the following categories:

- **Architecture feedback** — suggestions on project structure, component design, or data flow
- **UI/UX feedback** — usability observations, layout improvements, accessibility concerns
- **Security concerns** — potential vulnerabilities or information disclosure risks
- **Code quality** — readability, maintainability, TypeScript usage, or patterns
- **Testing ideas** — edge cases, test scenarios, or testing strategy suggestions

Please open a [GitHub Issue](https://github.com/kalaba992/Carinski-ekosistem-demo/issues) with your feedback.

This is a sanitized public demo. Issues related to the full production system or requests for access to the private repository are out of scope.
