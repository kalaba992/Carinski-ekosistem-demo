# Security

This repository is a **sanitized public demo** of the Carinski Ekosistem project.

- No secrets, credentials, API keys, or tokens belong in this repository.
- If you discover any suspected secret, credential, or sensitive information in this repo, please report it immediately through [GitHub Issues](https://github.com/kalaba992/Carinski-ekosistem-demo/issues).
- The private production repository (`Carinski-ekosistemm`) contains sensitive business and customs data. Access to that repository is not required for public review of this demo.
- Do not commit `.env`, `.env.*`, or any configuration file containing real credentials.
